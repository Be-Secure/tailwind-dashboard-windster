Windster is a premium Tailwind CSS dashboard template featuring over 21 application pages, advanced charts, a kanban board, an expandable sidebar layout, SVG maps, authentication pages, and many more.

This product is based on the popular open source Flowbite component library built with the utility classes from Tailwind CSS and it also uses HUGO to statically generate the HTML files and Webpack to bundle together the asset files.

Build with Tailwind CSS

This dashboard template is built using the utility classes from Tailwind CSS and you can work with this dashboard with the same development flow that you’ve been accustomed to when working with Tailwind CSS.

Hundreds of Tailwind CSS components are also included in this dashboard including modals, datepickers, dropdowns, buttons, and many more based on the Flowbite component library which is included by default as a plugin.

The final CSS file is automatically purged and minified based on the Webpack and PostCSS configuration file that we’ve developed. The classes that you see in the demo version are encrypted, however, when you purchase the product you will unlock the Tailwind CSS classes. This is to prevent stealing the code with the developer tools.

HUGO static site generator

Although you can use Windster any way you see fit we decided to use the HUGO static site generator engine to develop the website layout and show data using JSON files. This makes it easier to see which blocks of code are being reused in multiple pages such as the sidebar, the navigation bar, and the footer.

In the readme file of this product you will learn more about configuring and building the HUGO codebase. Of course, you can also adapt the HTML layouts for your own project whether you use Laravel or other back-end frameworks.

Webpack bundler

Webpack is one of the most advanced assets bundlers out there and in combination with PostCSS we found the perfect configuration for the Tailwind CSS workflow and assets bundling. You can use the Webpack config file in any way you want and of course you can also customize it based on your project needs.

21+ Admin Dashboard Pages

Windster is supposed to save you time developing an admin dashboard layout by having a wide collection of fully-coded pages such as the main dashboard layout, users page, products page, settings page, messages, kanban board, authentication pages, and many more.

You can check out the full list of pages by scrolling through this product page or by viewing the live demo and browsing the sidebar menu on the left side.

Kanban board
